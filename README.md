# 项目说明

项目使用 [poseidon-boot-starter](https://github.com/muggle0/poseidon-boot-starter) 脚手架基于 `springboot 2.0.6.RELEASE` 进行开发 

项目使用的框架：

- quartz
- flowable
- spring-security
- spring-stateMachine 
- netty
- webflux

项目结合实用性与学习实践的角度进行开发功能
